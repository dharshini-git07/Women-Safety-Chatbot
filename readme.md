# Women Safety Awareness Chatbot

This project is an AI chatbot that provides women safety awareness tips, emergency helpline numbers, and guidance.

## Technologies Used
- Python
- Streamlit
- Ollama
- Phi-3 Model

## Installation

1. Install Ollama
2. Pull the model
ollama pull phi3

3. Install requirements
pip install -r requirements.txt

4. Run the application
streamlit run app.py

## Features
- Women safety awareness tips
- Emergency helpline numbers
- AI-powered responses
